# laiyu64.github.io

`laiyu64` 的 GitHub Pages 个人主页。

## 本地预览

这是纯静态页面，直接双击 `index.html` 就能预览，或者用任意静态服务器打开。

## 发布方式

1. 在 GitHub 新建仓库：'https://laiyu64.github.io/laiyu64-github.io/'
2. 把当前目录文件推送到该仓库默认分支
3. 在仓库 `Settings -> Pages` 中确认已启用 GitHub Pages
4. 访问 `https://laiyu64.github.io/`
