GitHub 个人主页

这个页面记录我搭建个人主页的过程。相比普通的静态页面，我希望这个站点能长期维护，用来整理大学资料、项目实践、算法训练和日记。

项目定位

这个主页的定位是“大学生技术成长主页”，不是单纯的个人简介页。它既可以展示个人信息，也可以作为长期学习档案。

主要栏目

  • 个人简介：介绍基本信息、技能栈、学习方向和联系方式。
  • 大学资料：整理课程笔记、算法训练、竞赛资料和实验报告。
  • 项目集：记录个人项目、课程项目和后端开发实践。
  • 日记：记录学习过程、比赛复盘和阶段总结。

后续计划

  • 继续完善页面样式。
  • 增加更多真实文章内容。
  • 整理算法题解和项目复盘。
  • 让主页能够作为简历中的个人技术主页使用。
