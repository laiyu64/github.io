项目集 / TinyWebServer

TinyWebServer 项目记录

TinyWebServer 是我当前重点学习和实践的 C++ 后端项目。通过这个项目，我希望系统理解 Linux 网络编程、HTTP 请求处理、线程池、epoll、多路复用、数据库连接池和日志系统等内容。

项目学习目标

  • 理解浏览器请求是如何进入服务器的。
  • 理解 socket、bind、listen、accept 的基本流程。
  • 理解 epoll 在高并发连接处理中的作用。
  • 理解线程池如何处理客户端请求。
  • 理解 HTTP 请求解析和响应生成过程。
  • 理解 MySQL 在注册登录功能中的使用方式。
  • 理解日志系统和定时器的作用。

当前进展

目前已经完成项目环境搭建、基础编译运行、端口监听和浏览器访问测试。后续会继续整理源码阅读笔记，并尝试进行二次开发。
